Integration of R2lab wireless tesbed with ns-3 network simulator



I worked on this project as a part of my internship at INRIA, France. I successfully integrated the R2lab wireless testbed with ns-3 network simulator. MANET routing protocols were tested on various emulated network topologies using tools such as VLC (for both multicast and unicast streams), mcsender and ping. I also wrote nepi-ng scripts to automate image loading, network configuration, packet capturing, result collection while testing network protocols.


   

Corrosion Damage Identification and Lifetime Estimation of Ship Parts using Image Processing



I worked on this project as a part of "Computer Graphics and Mini Project" course. Working as part of a team, I assessed corrosion damage in ship parts and predicted their lifetime (when both pre-corrosion and post-corrosion ship images are present). In the presence of only post-corrosion ship images, we used SLIC (Simple Linear Iterative Clustering) for image segmentation and classified the segments as corroded or uncorroded using Random Forest Classifier. We also compared the performance of various corrosion prevention measures.


   

Extension of AQM Evaluation Suite for ns-3



I worked on this project as a part of "Computer Networks Lab" course. AQM Evaluation Suite automates the cycle of simulation setup, topology creation, traffic generation, program execution, results collection and graphical representation for evaluation of ns-3 queue disciplines. I extended the suite by enabling Byte Queue Limits (BQL) support and building API to remove undesirable AQM algorithms during evaluation.

Dimensionality Reduction using Linear Discriminant Analysis



As a part of this project, I implemented Linear Discriminant Analysis (LDA) in Python and compared its results with those obtained using the inbuilt function provided by scikit-learn. I also evaluated the performance of LDA by using it as a dimensionality reduction tedchnique prior to classification using Support Vector Machine (on Arcene and Madelon datasets). I later compared its performance with that of Principal Component Analysis (PCA).

Textile Waste Damage Analysis using Artificial Neural Network



I worked on this project as a part of "Soft Computing" course. Working as part of a team, I analyzed the general trend of the waste produced by the textile industries, in terms of the amount of fabric wasted, and predicted how much will be wasted in the future. We used Fuzzy Sets to perform feature selection and then designed an artificial neural network using Predictive Regression Model (optimized using Particle Swarm Optimization). This analysis enables industries to make a smarter choice in terms of raw materials they need and hence minimize textile wastes and subsequently, environmental pollution.


   

Implementation of TCP Prague in ns-3



I am working on this project as a part of my Bachelors thesis. Data Center TCP (DCTCP) provides low latency, low loss and scalable throughput in controlled environments like Data Center Networks. DCTCP cannot be deployed in public Internet as it uses a modified Explicit Congestion Notification (ECN) which may lead to performance and safety issues. TCP Prague modifies a number of features that make DCTCP unable to be successfully deployed in public networks. I am currently implementing TCP Prague in ns-3.
